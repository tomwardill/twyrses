Description

Twyrses is an ncurses based twitter client. For using twitter in a shell, without the expense of a nasty windowing system. It's fairly basic, but will let you view tweets, post tweets and some other functions.

Written in python and urwid, using tweepy as the twitter backend, mostly conceived with the assitance of alcohol, and expanded through 'well-I've-started-now-ness'.

Installation

At the moment, there are no packages available. If you have the dependencies (tweety, urwid) installed, then this link might work: Latest Twyrses

However, the most reliable way to run Twyrses is to follow the contributing instructions below, and obtain a code checkout.

Usage

Start Twyrses with 'python twyrses.py'.

To post a tweet, just type, and press Enter

Commands are prefaced with a '/'.

Command Result
/r Refresh your timeline, or return to the timeline
/@ View mentions (or @replies)
/d View Direct Messages
/s 'term' Search for 'term'
/t Set theme. Currently available: Default, White, Black
/f 'twittard1' 'twittard2' Check if twittard1 follows twittard2
/q Quit
/unfollow Unfollow a friend twittard
/follow Make a twittard a friend
