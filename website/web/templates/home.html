Description

Twyrses is an ncurses based twitter client. For using twitter in a shell, without the expense of a nasty windowing system. It's fairly basic, but will let you view tweets, post tweets and some other functions.

Written in python and urwid, using tweepy as the twitter backend, mostly conceived with the assitance of alcohol, and expanded through 'well-I've-started-now-ness'.
